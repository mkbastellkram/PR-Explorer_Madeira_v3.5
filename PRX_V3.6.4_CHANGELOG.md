# PR-Explorer Madeira V3.6.4 POI MAP DOCK

## Geändert
- POI-Auswahl fährt die PR-Detailkarte wie beim Journal/PR-Fluss nach links aus dem Sichtbereich.
- POI-Karussell wird als kompakter Dock im unteren Kartenbereich angezeigt.
- Kein separates POI-Sheet als primärer Zielzustand.
- POI-Karussell bleibt horizontal bedienbar; vertikales Runterziehen schließt zurück zur PR-Detailkarte.
- GPX-/KML-Kontext der aktiven PR bleibt erhalten.
- Aktiver POI wird auf der Karte hervorgehoben; übrige POIs der geöffneten Kategorie erscheinen dezent als Ghost-Marker.
- Grauer Griff-/Balkenbereich der Detailkarte reduziert.
- Peek-Endlage unten leicht entschärft, damit die Detailkarte nicht hart am Displayboden anschlägt.
- Version und Cache-Parameter auf 3.6.4 gesetzt.

## Nicht geändert
- Keine neuen POI-Daten.
- Keine Änderung an PR-/GPX-/KML-/POI-Rohdaten.
- Keine automatische Tagesroutenoptimierung.
- Keine Live-Statusabfrage.
- Keine neue Navigationsarchitektur.
- Keine Service-Worker-Änderung.

## Hinweis
V3.6.4 ist ein Bedienfluss-Patch für die POI-Auswahl. Ziel ist der gleiche räumliche Übergang wie bei der PR-Auswahl: Detailkarte fährt links aus, Karte wird frei, POI-Karussell liegt unten über der Karte.

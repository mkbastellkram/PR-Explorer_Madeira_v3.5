# PR-Explorer Madeira V3.6.3 POI SAME DETAIL FLOW

## Geändert

- POI-Darstellung im PR-Detailbereich repariert.
- Keine separaten POI-Sheets mehr als Bedienziel.
- POIs bleiben im gleichen Detailkarten-Aufbau wie die PR-Details.
- Kontextbereich auf kompakte vertikale Struktur umgestellt:
  - Sehenswürdigkeiten / Highlight-POIs als PR-ähnliche kompakte Zeilenkarten.
  - Funktions-/Info-POIs als Kategoriezeilen mit Anzahl.
- Antippen oder Links-Wischen einer Highlight-POI-Zeile öffnet ein eingebettetes horizontales POI-Karussell im Kontextbereich.
- Antippen oder Links-Wischen einer Funktionskategorie öffnet ein eingebettetes Kategorie-Karussell im Kontextbereich.
- Aktiver POI wird weiterhin auf der Karte markiert.
- GPX-/KML-Kontext der aktiven PR bleibt sichtbar.
- Ghost-Marker für weitere POIs der geöffneten Kategorie bleiben erhalten.
- Große, leer wirkende POI-Kacheln im Kontextbereich entfernt.
- Version und Cache-Parameter auf 3.6.3 gesetzt.

## Nicht geändert

- Keine neuen POI-Daten.
- Keine Änderung an PR-/GPX-/KML-/POI-Rohdaten.
- Keine automatische Tagesroutenoptimierung.
- Keine Live-Statusabfrage.
- Keine neue Navigationsarchitektur.
- Keine Service-Worker-Änderung.

## Hinweis

V3.6.3 ist ein Reparaturpatch für die POI-Bedienstruktur. Ziel ist, POIs im gleichen Detailkarten-Aufbau wie PRs zu führen und keine separaten Sheets zu verwenden.

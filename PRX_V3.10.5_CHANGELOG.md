# PR Explorer Madeira V3.10.5 – Copy Feedback & Audit Control Polish

## Geändert
- Kopierschalter geben jetzt sichtbares Feedback.
- Toast/Puls-Bestätigung für Audit kopieren, Diagnose kopieren und weitere Copy-Aktionen ergänzt.
- Clipboard-Feedback auch für programmatische Kopiervorgänge ergänzt.
- Audit-/Recorder-Schalter erhalten zusätzliche Komponentenkennungen, damit weniger Ereignisse als „unbekannt“ im Auditbericht erscheinen.
- Versionsanzeige defensiv auf 3.10.5 synchronisiert.
- Auditbericht wird defensiv auf Version 3.10.5 gepatcht und enthält einen Hinweis zum Copy-Feedback.

- Keine PR-/GPX-/KML-/POI-Rohdaten geändert.
- Keine Kartenarchitektur geändert.
- Kein Service Worker.
- Keine neue Ordnerstruktur.
- Keine Hotspot-/Heatmap-Integration.
- Keine automatische Tagesroutenoptimierung.

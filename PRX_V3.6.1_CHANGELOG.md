# PR-Explorer Madeira V3.6.1 POI CAROUSEL

Änderungen:
- POI-Bereich von vertikaler Kartenliste auf horizontales POI-Karussell umgestellt.
- POI-Karussell blockiert vertikales Mitscrollen innerhalb der POI-Schiene und nutzt horizontales Scroll-/Swipe-Verhalten.
- Aktive POI-Karte wird optisch markiert.
- Wischen/Scrollen im POI-Karussell wählt den mittig eingerasteten POI.
- Ausgewählter POI wird auf der Karte gezeigt, während GPX/KML-Kontext der aktiven PR erhalten bleibt.
- Zusätzlich werden die übrigen vorgeschlagenen Kontext-POIs dezent als Ghost-Marker gezeigt.
- POI-Aktionslogik Heute/Später/Erledigt/Maps/Ausblenden bleibt erhalten.

Nicht geändert:
- Keine neue Navigationsarchitektur.
- Keine automatische Routenoptimierung.
- Keine Änderung an PR-/GPX-/KML-/POI-Rohdaten.
- Keine Live-Statusabfrage.

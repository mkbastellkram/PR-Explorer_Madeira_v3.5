# PR Explorer Madeira V3.5.0

Neustart-Prototyp auf Basis des Scroll-Sheet-Dock-Modells.

## Prinzip

- Start im Journal
- PR-Kacheln scrollen als normale Safari-Webseite
- Karte startet erst nach dem ersten Tipp auf eine PR
- Floating Dock statt starrem Bottom-Balken
- Dock parkt beim Scrollen links ein
- Kein Service Worker in dieser Testversion
- Audiodatei optional: `assets/audio/madeira_intro_ambience.mp3`

## Upload

Empfohlen als neues Repository:

`PR-Explorer-Madeira_v3.5`

GitHub Pages:
Settings → Pages → Deploy from a branch → main → /(root)
